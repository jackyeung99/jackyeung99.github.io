This project consisted of two major components: data engineering and machine learning modeling.

Data Engineering

I built a data pipeline to retrieve publicly available apartment listings using web scraping. I developed bots to scrape HTML content for listings in a given city, parsing the extracted data and storing it as a JSON file. To structure the data effectively, I designed an Entity-Relationship Diagram (ERD) and implemented a SQLite3 relational database, ensuring robust data integrity through Pydantic for validation. This approach was necessary to ensure accurate predictions at scale.

Machine Learning Modeling & Analysis

The second phase of the project focused on analyzing the data and developing predictive models. Through Exploratory Data Analysis (EDA), I identified trends and challenges in real-world data quality. I then built three machine learning models:

  • XGBoost (Extreme Gradient Boosted Trees)
  • Standard Neural Network
  • Lasso Regression

Given the limited numerical data available from HTML content, I sought to enhance model accuracy by incorporating text-based features.
